<!-- ******************************Disney Characters App****************************************
This application is a simple web app that displays a list of Disney characters and allows the user to search and filter the list. The app also allows the user to navigate to a details page for each character, where they can see additional information about the character, such as their films, TV shows, and video games.

******************************Getting Started******************************
To run this application locally, follow these steps:

Clone the repository to your local machine
Open the project in Visual Studio
Build the solution to restore any missing dependencies
Run the application using IIS Express
Technologies Used
This application was built using .NET 6.0 and Blazor Server. It also uses the Disney API to retrieve data about the Disney characters.

******************************Features******************************
Display a list of Disney characters
Filter the list by name
Navigate to a details page for each character
See additional information about each character, such as their films, TV shows, and video games
